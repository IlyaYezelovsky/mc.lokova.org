LokovaMC 是一个真正以玩家为中心的 Minecraft Java 版新概念插件生存服。

我们具有更方便的插件系统、更少的管理规则、更友好的游戏特性、更开放的玩家社区、更与 Mojang 隔离的社交环境。

当你受不了某些服务器的传送插件有读秒和 CD、创建箱子商店需要消耗货币、没法使用 TNT 复制、有假矿……那么你来对地方了！

当然，如果你不在意上述那些，也可以来这里——我们同样欢迎！

服务器内没有规定玩家「该怎样玩」的限制——生电、建筑、养老、探险、PvP，你愿意怎么玩就怎么玩！

甚至——如果你认为「前期挖矿死了怎么办，大半个背包的铁都没了」，就可以关闭死亡掉落；如果认为「要掉落才有挑战」，就可以开启！这对每个玩家独立生效。还有 PvP 也是——如果你不想被别人突然袭击，就可以关掉 PvP——这样既不能被打，也不能打人。

LokovaMC 使用 Leaves 服务端，进行了多项有利于生电及普通生存的特性修改，详见 [配置](/config)。
